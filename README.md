# ACQA - JavaScript

Este repositório contém a resolução da Avaliação Continuada de Questões Abertas (ACQA) da disciplina de JavaScript.

## 📚 Objetivo

Aplicar os conhecimentos de JavaScript, utilizando:
- prompt
- alert
- confirm
- Arrays
- Estruturas de repetição
- Manipulação do DOM

---

## 📁 Estrutura do projeto

- `questao1.html` → Caixa de confirmação
- `questao2.html` → Nome do mês usando Array
- `questao3.html` → Números pares, soma e contagem
- `questao4.html` → Lista de nomes com ordenação e transformação

---

## 🧠 Questões desenvolvidas

### ✔ Questão 1
Exemplo de uso de caixa de confirmação com `confirm()`.

---

### ✔ Questão 2
Entrada de número do mês via `prompt` e exibição do nome por extenso utilizando Array de objetos.

---

### ✔ Questão 3
Solicita um número maior que zero e exibe:
- Números pares entre 1 e o número informado
- Quantidade de números pares
- Soma dos números pares

---

### ✔ Questão 4
Solicita uma quantidade de nomes e:
- Armazena em um Array
- Exibe na ordem de inserção
- Exibe em ordem alfabética
- Exibe todos em maiúsculo
